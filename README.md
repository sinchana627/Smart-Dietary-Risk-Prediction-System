# HealthML Frontend

React-based frontend for Smart Health Risk Prediction System.

## Quick Start

```bash
npm install
npm start
```

Runs on http://localhost:3000

## Pages

- `/` - Home page
- `/login` - User login
- `/signup` - User registration
- `/dashboard` - Main prediction interface

## Components

- Home - Landing page
- Login - User authentication
- Signup - User registration
- Dashboard - Main application interface

## Features

- Responsive design
- Real-time predictions
- Prediction history
- Model performance visualization
- Risk assessment charts

## Technologies

- React 18
- React Router 6
- Axios - HTTP client
- Recharts - Data visualization
- CSS3 - Styling
